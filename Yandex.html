Yandex
    Historia

    Yandex Browser es un navegador web gratuito desarrollado en 2012 para dispositivos móviles y ordenadores por la corporación rusa de búsqueda web1​ que utiliza el motor de renderizado Blink utilizado en el proyecto de código abierto Chromium. El navegador comprueba la seguridad de la página web con el sistema de seguridad Yandex y comprueba los archivos descargados con el antivirus Kaspersky. El navegador también utiliza la tecnología Turbo de Opera Software para acelerar la navegación en conexiones lentas.
